  ## About this repository

  This repo hosts the official release builds of **BeemNotch** — the full
  macOS app distributed as a signed `.dmg`. The source code is closed-source
  and not published.

  Use the [Releases](https://github.com/BeemNotch/Releases/releases) page to
  download the latest version. Each release includes a changelog and the
  installer.

  ## Requirements

  - macOS 12 (Monterey) or later
  - MacBook with a notch recommended — works on other Macs with fallback positioning
  - Apple Silicon or Intel

  ## Installation

  1. Download `BeemNotch.dmg` from the [latest release](https://github.com/BeemNotch/Releases/releases/latest)
  2. Open the DMG and drag **BeemNotch** into `/Applications`
  3. Launch from Spotlight or the Applications folder
  4. Grant the permissions it asks for (Notifications, Calendar, Microphone for Voice Bridge, etc.)
